# UMaT Postgraduate Administrative Support System (UPASS)

## Getting Started

### For New Members (After Cloning)

1. **Run `sync.bat`** — Syncs environment variables and initial configuration
2. **Run `export_schema.js`** — Creates the database schema (tables, indexes, etc.)
3. **Run `run_migrations.js`** — Applies any pending migrations

```bash
# From the project root:
sync.bat
node backend/export_schema.js
node backend/run_migrations.js
```

### For Existing Members

#### Creating a Superadmin
## Run all batch scripts from the root dir with a cmd terminal
```bash
create-admin.bat
```
This creates the initial superadmin user account.

#### Clearing the Database
```bash
clear-database.bat
```
**Warning:** This deletes all data. After clearing, you must re-run:
```bash
node backend/export_schema.js
node backend/run_migrations.js
```

```
│   │   ├── controllers/      # Route handlers

│   ├── migrations/           # SQL migration files
│   └── sample_*.xlsx         # Sample bulk upload files
├── src/                      # React frontend (Vite + TypeScript)
│   ├── pages/                # Page components by role
│   ├── components/           # Reusable UI components
│   └── lib/                  # Utilities (export, API client)
├── sync.bat                  # Environment/config sync script
├── create-admin.bat          # Create superadmin user
├── clear-database.bat        # Wipe database (use with caution)

```

## Tech Stack

- **Frontend:** React + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend:** Node.js + Express + PostgreSQL
- **Auth:** JWT-based authentication with role-based access control
- **PDF Export:** jsPDF with custom table rendering

## Available Roles

- **Student** — Submit thesis, view results, track fees, request documents
- **Supervisor** — Review thesis submissions, provide feedback
- **Accountant** — Manage fees, generate financial reports
- **Admin** — Manage users, students, courses, departments
- **Dean / ViceDean** — Approve clearances, view CWA results
- **Registrar** — Manage results, pass lists, exam timetables
- **Exams Officer** — Publish results, generate pass lists
- **Admin Assistant** — Support admin operations

## Key Features

- Thesis submission & supervisor review workflow
- Fee management with bulk upload (CSV/Excel)
- Student clearance approvals (multi-step)
- CWA (Cumulative Weighted Average) results & analytics
- Document request management
- Financial report exports (CSV & PDF)
- AI assistant for supervisor feedback
- Real-time notifications

## Development

```bash
# Install dependencies (root + backend)
npm install
cd backend && npm install && cd ..

# Start backend (port 5000)
cd backend && npm run dev

# Start frontend (port 5173)
npm run dev
```
